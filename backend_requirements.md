## 三、每日报告数据

> [待编写] `/daily/report` —— 按日期返回当晚睡眠+心脏+呼吸完整报告

### 需要后端提供的数据结构（待细化接口格式）：

#### 3.1 摘要卡
- 身体恢复分 (0-100)
- 精神恢复分 (0-100)
- 综合评分 (0-100)
- 睡眠时长

#### 3.2 核心健康指标（7 项）
- 睡眠时长、睡眠效率、深睡占比
- 平均心率、HRV、AHI、最长呼吸暂停

#### 3.3 睡眠分期
- 分期数据点（清醒/REM/浅睡/深睡，每 5 分钟一个点）
- 睡眠循环（每轮起止时间 + 各期占比 + 累计恢复分）
- 分期图例（各阶段时长和百分比）

#### 3.4 心脏健康
- 心脏健康指数 (0-100)
- 心率范围
- HRV 值、平均心率、rMSSD
- 心率趋势序列（96 个数据点）
- ECG 波形序列（48 个数据点）

#### 3.5 呼吸健康
- 呼吸健康指数 (0-100)
- 呼吸暂停统计（次数/最长/平均）
- 呼吸频率趋势序列（100 个数据点）

#### 3.6 体动数据
- 翻身次数
- 体动趋势序列（48 个数据点）

#### 3.7 声音监测
- 鼾声/梦话/咳嗽 发生次数与时长
- 声音录音片段列表（时间、时长、类型）

#### 3.8 风险提示（4 项）
- HRV、呼吸暂停、夜间心率等风险评估
- 每项含：状态（绿/黄/红）、数据解读、长期风险因素、改善建议

#### 3.9 每日建议
- 入睡建议时间
- 起床建议时间
- 今日打卡项（3 项）

## 四、长期趋势数据

> [待编写] `/trend/{metric}` —— 按指标返回 7/30/90 天时间序列

### 13 项指标：

| 指标 | 单位 | 需要的数据 |
|------|------|-----------|
| 睡眠时长 | 小时 | 每日值 + 14 日均值 |
| 睡眠效率 | % | 每日值 + 14 日均值 |
| 入睡时长 | 分钟 | 每日值 + 14 日均值 |
| 深睡占比 | % | 每日值 + 14 日均值 |
| 平均呼吸频率 | 次/分 | 每日值 + 日最大/最小值 + 14 日均值 |
| AHI | 次/时 | 每日值 + 14 日均值 |
| 最长呼吸暂停 | 秒 | 每日值 + 14 日均值 |
| 夜间平均心率 | bpm | 每日值 + 日最大/最小值 + 14 日均值 |
| HRV | ms | 每日值 + 日最大/最小值 + 14 日均值 |
| 心脏健康指数 | 分 | 每日值 + 14 日均值 |
| 鼾声次数 | 次 | 每日值 + 14 日均值 |
| 梦话次数 | 次 | 每日值 + 14 日均值 |
| 作息时间 | 时分 | 每日上床/起床时间 + 均值/规律性统计 |

### 额外统计
- 各模块汇总评分（睡眠/呼吸/心脏/声音 四模块）
- 心脏年龄推算（基于 HRV、静息心率、实际年龄）
