# 话好眠 — 心脏健康 × 睡眠监测 微信小程序

「话好眠」是一款基于微信云开发的健康监测小程序。最初定位为睡眠健康监测，本期开发后**核心定位调整为心脏健康监测**，睡眠作为夜间监测心脏状态的载体。

- **AppID**: `wx399b0d556538592e`
- **基础库**: 2.20.1
- **小程序根目录**: `miniprogram/`
- **云函数根目录**: `cloudfunctions/`

---

## 一、页面结构

| 路径 | Tab 名称 | 角色 |
|---|---|---|
| `pages/daily/index` | 每日 | 当日睡眠期间的心脏 / 呼吸 / 睡眠综合报告 |
| `pages/longterm/index` | 长期 | 多维度趋势分析（心脏、呼吸、睡眠三大板块） |
| `pages/advice/index` | 建议 | 风险卡 + 打卡进度 + 健康改善建议 |
| `pages/my/index` | 我的 | 设备管理 + 监测设置 + 预警 + 物联卡 |
| `pages/temple/index` | — | 好眠寺辅助页（从每日页跳转） |

## 二、每日 Tab（pages/daily）

### 板块顺序（重构后）

1. **心脏健康** —— 默认**展开**
2. **呼吸健康** —— 默认折叠
3. **睡眠健康** —— 默认折叠（含好眠寺入口 / 时间轴 / 恢复双环 / 睡眠分期 / 体动健康）
4. **声音监测** —— 默认隐藏
5. **风险提示** —— 始终显示（HRV / 呼吸暂停 / 夜间心率）
6. **改进建议** —— 始终显示（提升 HRV、改善呼吸暂停、降低静息心率、长期心脏追踪）

### 心脏健康板块

- 心脏综合卡：含心脏健康指数渐变条（`hi-bar`，红→白→蓝五段渐变，黑色圆形指针），状态收起/展开
- 2×2 指标网格：心率范围 / HRV值 / 平均心率 / rMSSD
- 心率数据卡：折线趋势图 + "未见异常"胶囊 + 蓝色分析框（含可点击术语 tooltip）
- 心动图卡：ECG 波形 + 橙色分析框（P-QRS-T、窦性心律解读）

### 呼吸健康板块（与心脏板块结构一致）

- 呼吸健康主卡：折叠头 + 副标题
- **呼吸健康指数卡**：渐变条指示（`#58c7f5` 蓝色指针，落在优秀区）+ 3 项暂停统计（次数 / 最长 / 平均）+ 橙色分析框（与同龄人对比、AHI 评估、风险阈值）
- **呼吸频率卡**：100 个数据点的密集折线图（Y 5-25，平均 16 次/分）+ 蓝色分析框（频率范围、稳定性、波动原因）

### 睡眠健康板块

- 副标题动态显示：`昨晚睡眠 8h36m，身体恢复87分，精神恢复87分`
- 内嵌内容（展开时一次性绘制全部 canvas）：
  - **好眠寺入口**（圆形按钮）
  - **时间轴 + 双环恢复度**（前晚 / 昨晚 / 建议今晚 三列对照 + 身体/精神恢复 conic-gradient 双环）
  - **简短睡眠评价 + sleepWins 3 项小卡**
  - **🛏️ 睡眠分期**：起止时间、Canvas 分期图、6 段图例、5 个睡眠循环条 + 累计恢复%
  - **🏃 体动健康**：翻身次数 + 体动趋势小图 + 图例

### 状态字段

- `showHeart` （默认 true）
- `showBreathing` / `showSleepHealth` / `showMovement` / `showAdvanced`（均默认 false）

## 三、长期 Tab（pages/longterm）

### 模块结构

| 模块 | 默认状态 | 包含图表 |
|---|---|---|
| 睡眠健康 | 折叠 | 作息柱形图 + 睡眠时长 / 效率 / 入睡时长 / 深睡占比 |
| 呼吸健康 | 折叠 | 平均夜间呼吸频率 + AHI + 最长呼吸暂停时长 |
| 心脏健康 | **展开** | 夜间平均心率 + HRV 心率变异性 + 心脏健康指数 |

每个模块卡都有：emoji + 标题 + 评分胶囊（excellent/good/warn 三色）+ 指标项统计 + 重点关注。

### 核心图表系统

所有图表使用 **微信 Canvas 2D API** 原生绘制，无第三方库。

**`drawHealthLineChart`** — 折线图（适用于睡眠时长、效率、入睡时长、深睡比、AHI、最长呼吸暂停、HRV、心脏健康指数）：
- 健康分区色带（健康区/亚健康/不健康），左侧带分区标签胶囊
- 14 天均值虚线（深色）+ 同龄人参考线（灰色虚线）
- Catmull-Rom → Bezier 三次曲线平滑（解决了"曲线 + 直线相间"的旧 bug）
- 数据点支持绿点（建议见效日）/ 红点（警示日）双色
- annotation 标注气泡：上方深色胶囊 + 虚线引导至对应数据点
- 点击数据点弹深色 tooltip（数值 + 日期 + 三角尾巴）
- Y 轴范围根据每个指标收窄到合理区间（如 AHI 0-10、HRV 10-140、深睡 5-32）

**`drawHealthBandChart`** — 区间柱形图（适用于夜间平均心率、HRV、平均夜间呼吸频率）：
- 每日 min-max 浮动胶囊柱
- 14 天均值 + 同龄人参考线
- 健康分区色带 + annotation 气泡

**`drawScheduleBarChart`** — 作息柱形图：
- Y 轴 22:00 在底部 / 10:00 在顶部
- 统一淡蓝色柱体，上下端无缺口
- 平均上床时间已修复跨日 bug（00:43 之类的时间不再被算成 43 分钟）

### 交互能力

- 全部图表支持周 / 月 / 全周期三档切换
- 月/全周期可使用 `« ‹ › »` 4 向导航
- 触摸手势：左右滑动可切换时间窗
- 图例统一 5 项：健康区 / 亚健康 / 不健康 / 14天均值 / 同龄人

### 修复的关键 bug

1. **avg14Day 重合 bug**（AHI / 最长呼吸暂停 / 心脏健康指数）：原本 `series.map(v => v * 0.85)` 把数据线乘系数生成"假均值线"，结果均值线与数据线几乎重合。修复为从专用平滑趋势数组（`ahiAvg14` / `apneaAvg14` / `heartIndexAvg14`）取数，呈现独立趋势走向
2. **作息柱形图 Y 轴反转**：原本 22:00 在顶、10:00 在底，已倒置
3. **柱形图白色缺口**：移除 `arc()` 端点白点
4. **avg 跨日时间错算**：归一化算法 `v >= 18*60 ? v : v + 24*60`，求平均后再 `% 24*60`
5. **折线"曲线 + 直线相间"**：用真正的 Catmull-Rom → Bezier 转换
6. **健康分区遮挡空白**：每个指标收窄 yMin/yMax 范围

## 五、我的 Tab（pages/my）

### 架构

单页多视图模式（`currentView` 状态机），避免在 `app.json` 注册多个子页：
- `main` —— 设备名 + 7 项菜单（无 tab 切换）
- `adapt` —— 设备自适应
- `params` —— 监测参数设置
- `user` —— 被监测人信息
- `sim` —— 物联卡（ICCID/运营商/流量/有效期）
- `alert` —— 预警设置（3 段：心率 / 呼吸 / 离床）
- `share` —— 分享设备
- `more` —— 更多设置
- `device` —— 设备详情（圆形设备插画 + 名称 + 固件 + 红色"解除绑定"按钮）
